# 📘 Assignment: 🧮 Simple Calculator

## 🎯 Objective

Students will build a basic command-line calculator in Python to practice input/output, arithmetic operations, and error handling.

## 📝 Tasks

### 🛠️  Implement calculator operations

#### Description
Write functions that perform addition, subtraction, multiplication, and division based on user input.

#### Requirements
Completed program should:

- Prompt the user for two numbers and an operation symbol (`+`, `-`, `*`, `/`)
- Call the appropriate function to perform the calculation
- Print the result with an informative message
- Handle division correctly (e.g., `7 / 3` should result in `2.333...`)


### 🛠️  Add input validation and error handling

#### Description
Ensure the program gracefully handles invalid inputs and edge cases.

#### Requirements
Completed program should:

- Display an error message if the user enters a non-numeric value
- Prevent division by zero and inform the user when it occurs
- Prompt the user again if they choose an unsupported operation symbol
- Offer the user the ability to perform another calculation or exit
